| comando                                                   | funcion                                |
|-----------------------------------------------------------|----------------------------------------|
| node -v                                                   | version de Node js                     |
| npm install -g @angular/cli                               | instalar ultima version de angular     |
| ng version                                                | version instalada de angular/cli       |
| cls                                                       | para limpiar la consola                |
| git init                                                  | iniciar repositorio                    |
| git status                                                | verificar estado de mis archivos       |
| git add                                                   | sube todos los archivos al git         |
| git commit                                                | sellamos los archivos para enviar      |
| git remote add origin link                                | enlazar el repositorio                 |
| git remote -v                                             | verificar repositorio                  |
| git push                                                  | para enviar al repo remoto             |
| ng new mi-primer-angular--standalon --routing --style=css | crear proyecto en angular              |
| ng serve -o                                               | corre el servidor y abrir el navegador |
| ng generate(g) component(c) view/user/user.component      | Crea componentes para vistas           |
